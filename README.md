Service Provider Platform

A full-featured web application for connecting customers with local service providers (plumbers, electricians, cleaners, tutors, etc.). Users can search for providers, view detailed profiles, read reviews, and book appointments. Providers can register, manage their profiles, respond to booking requests, and communicate with customers.

**Features:**
- Provider and user registration/login
- Provider profiles with services, pricing, and reviews
- Booking system with calendar and time slots
- Customer reviews and ratings
- Responsive design (Bootstrap)
- Demo data for providers and reviews

This project was built from scratch as an original idea using HTML, CSS, JavaScript, and Bootstrap. It showcases real-world skills in UI/UX, form validation, modal workflows, and multi-role dashboards.
